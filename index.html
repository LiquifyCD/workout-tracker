Rotation logic

Lats/back: Day 1 uses wide pulldown + row, Day 3 repeats shoulder adduction, Day 5 uses single-arm lat bias.
Chest: Day 1 pec deck, Day 3 incline press + pec deck, Day 5 pec deck only because overhead press already loads front delts.
Hamstrings: hinge + curl, curl-only priority, hinge + curl.

Rules

Structure: upper/lower loop. No fixed rest day. Rest only when recovery markers drop.
Volume: 1–2 hard sets per exercise. Never 3+ sets for the same exercise in one workout.
Reps: 4–7 for stable compounds, 6–10 for isolations, 8–15 for lateral raises, 6–12 for calves.
Progression: increase when all working sets hit the top of the range with ≤1 RIR. Repeat if not maxed. Reduce/check recovery if below the range.
Rest triggers: performance drops two sessions in a row, joints ache, sleep is poor, motivation is flat, or normal loads feel like grinding.
